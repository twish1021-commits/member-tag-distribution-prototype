# 会员标签分布 需求文档

## 1 标签分类规则

### 1.1 分类定义

- **系统标签**：系统内置，不可删除，值由系统定义，多值枚举型（≥2个值）
- **个性化标签**：客户自行创建/删除/启用/禁用，值为固定的二值枚举 `["符合", "不符合"]`，标签名由客户动态创建，数量不固定

### 1.2 系统标签枚举清单

```json
{
  "system_tags": [
    {"name": "性别", "type": "fixed", "values": ["男", "女", "未设置"]},
    {"name": "会员级别", "type": "dynamic", "values": "根据系统设置的会员级别动态显示"},
    {"name": "年龄段", "type": "fixed", "values": ["儿童", "少年", "青年", "中年", "老年"]},
    {"name": "入会渠道", "type": "fixed", "values": ["门店", "微信小程序", "美团点评", "口碑", "淘宝", "抖音", "其它"]},
    {"name": "活跃度", "type": "fixed", "values": ["沉睡", "低活跃度", "活跃度中", "活跃度高"]},
    {"name": "消费总金额", "type": "fixed", "values": ["消费0元", "消费1k以内", "消费一两百", "消费两三百", "消费三四百", "消费五百起"]},
    {"name": "生日月份", "type": "fixed", "values": ["1月", "2月", "3月", "4月", "5月", "6月", "7月", "8月", "9月", "10月", "11月", "12月"]},
    {"name": "是否已绑定", "type": "fixed", "values": ["已绑定", "未绑定"]},
    {"name": "是否已关注", "type": "fixed", "values": ["已关注", "未关注"]},
    {"name": "出生年", "type": "fixed", "values": ["当年", "去年", "更早"]},
    {"name": "注册时间", "type": "fixed", "values": ["当月", "近3个月", "近半年", "更早"]}
  ]
}
```

字段说明：
- `type=fixed`：标签值为固定枚举，页面展示全部枚举值
- `type=dynamic`：标签值根据系统配置动态生成，非固定枚举
- 仅"性别"标签包含"未设置"值；其他系统标签不包含"未设置"，无值会员不计入任何标签值行

### 1.3 个性化标签值定义

```json
{"personal_tag_values": ["符合", "不符合"]}
```

## 2 业务规则

### 2.1 人数与占比计算

```
占比 = 标签值预估人数 / 当前数据范围会员总数 × 100%
```

| 参数 | 定义 |
|------|------|
| 预估人数 | 被打上该标签值的会员数，后端返回，前端只展示 |
| 分母（会员总数） | 当前数据范围的全部会员，包含有效和已过期状态会员 |
| 占比精度 | 四舍五入保留1位小数 |
| 多标签不互斥 | 同一会员可属于多个标签值，各标签值独立计数，总和可超过会员总数 |
| "未设置"展示规则 | 仅"性别"标签展示"未设置"值；其他系统标签不展示"未设置"，无值会员不计入任何标签值行 |
| 失效标签规则 | 接口不返回失效的个性化标签，页面不展示 |

### 2.2 排序规则

| 排序对象 | 排序方式 |
|----------|----------|
| 标签值行（表格内） | 按预估人数降序；人数相同时按标签值自然顺序 |
| 系统标签组内标签项 | 按系统定义顺序 |
| 个性化标签组内标签项 | 按创建时间倒序 |
| 分组顺序 | 系统标签组在前，个性化标签组在后，固定不变 |

### 2.3 数据刷新规则

| 触发条件 | 行为 |
|----------|------|
| 进入页面 | 自动加载一次，更新时间同步刷新 |
| 点击"刷新"按钮 | 重新加载所有数据 + 更新时间 |
| 自动轮询 | 不做，无定时刷新机制 |

### 2.4 数据范围自动判定

| 登录角色 | 数据范围 |
|----------|----------|
| 总部后台账号 | 总部全部会员 |
| 门店后台账号 | 当前分店会员 |

- 无门店选择控件，数据范围由登录组织自动决定
- 门店后台看到的总部标签数据可能存在延迟，底栏常驻提示

## 3 页面交互说明

### 3.1 页面布局结构

> 以下仅为结构示意，不含视觉规格。视觉样式参考 prototype_url。

| 区域 | 位置 | 内容 |
|------|------|------|
| A-顶部栏 | 页面顶部，单行横排 | 页面标题 → 搜索框 → 刷新按钮 → 更新时间 |
| B-左侧标签列表 | 左侧，纵向可滚动，独立滚动不影响右侧 | 标签列表标题 + 标签总数徽章；两个可展开/收缩分组（系统标签 / 个性化标签） |
| C-右侧分布详情 | 右侧，与B左右并列 | 标签信息区 + 分布表格；未选中标签时显示"请从左侧选择标签查看分布" |
| D-底栏 | 页面底部 | "跨门店数据可能存在延迟"，右对齐 |

区域关系：A横跨顶部全宽 → B与C左右并列占满中间 → D横跨底部全宽。

### 3.2 顶部栏元素

| 元素 | 功能 | 交互逻辑 |
|------|------|----------|
| 页面标题 | 显示"会员标签分布" | 只读 |
| 搜索框 | 输入关键词搜索标签 | 实时过滤左侧列表，匹配标签名或标签值，大小写不敏感 |
| 刷新按钮 | 手动刷新数据 | 点击后重新加载所有数据，更新时间同步刷新 |
| 更新时间 | 显示数据最后更新时间 | 格式："更新：YYYY-MM-DD HH:mm:ss"，刷新时同步更新 |

### 3.3 左侧标签列表

| 元素 | 说明 |
|------|------|
| 列表标题 | "标签列表"，右侧显示当前过滤后的标签总数徽章 |
| 分组标题 | "系统标签"/"个性化标签"，右侧显示分组标签数量；点击可展开/收缩；默认全部展开 |
| 分类标识 | 系统标签与个性化标签各有不同标识色，参考原型 |
| 标签项 | 标签名 + 该标签覆盖的预估人数合计；点击选中→右侧展示该标签分布详情；选中项高亮，参考原型 |

搜索过滤行为：

| 场景 | 行为 |
|------|------|
| 输入标签名 | 仅显示名称匹配的标签 |
| 输入标签值 | 显示包含该值的标签 |
| 当前选中不在过滤结果中 | 自动切换到第一个匹配标签 |
| 无匹配 | 列表显示"无匹配标签" |

### 3.4 右侧分布详情

标签信息区：

| 元素 | 内容 |
|------|------|
| 分类徽章 | 系统标签→"系统"徽章；个性化标签→"个性化"徽章，样式参考原型 |
| 标签名 | 当前选中标签的名称 |
| 预估人数 | "预估 N 人"，N为当前数据范围的会员总数 |

分布表格：

| 列名 | 列头文案 | 内容 | 对齐方式 |
|------|---------|------|----------|
| tag_value | "标签值" | 标签值名称 | 左对齐 |
| distribution | "分布" | 横向进度条，宽度=占比百分比 | 左对齐 |
| estimated_count | "预估人数" | 预估人数 | 右对齐 |
| percentage | "占比" | 占比值+"%" | 右对齐 |

- 表格行按预估人数降序排列
- 进度条样式参考原型
- 搜索关键词匹配某标签值名称时，该行背景高亮，参考原型

## 4 异常处理

| 场景 | 处理方式 |
|------|----------|
| 会员总数为0 | 所有标签值显示 0人 / 0.0%，右侧顶部提示"暂无会员数据" |
| 某标签无数据 | 左侧仍显示该标签项，右侧选中后所有值为0 |
| 搜索无结果 | 左侧显示"无匹配标签"，右侧显示引导提示 |
| 数据加载失败 | 保留旧数据不覆盖，提示"数据刷新失败，请稍后重试"，提示自动消失 |
| 跨门店数据延迟 | 底栏常驻提示，不做阻断 |
| 权限不足 | 隐藏页面菜单入口；强行访问显示"暂无权限" |
| 失效个性化标签 | 接口不返回，页面不展示 |

## 5 验收标准

### 5.1 功能验收

| ID | 验收项 | 标准 |
|----|--------|------|
| F-01 | 标签展示 | 系统标签与个性化标签全部展示，无遗漏 |
| F-02 | 预估人数与占比 | 各标签值预估人数与占比准确，保留1位小数；占比分母为全部会员（含有效和已过期） |
| F-03 | 分组展开/收缩 | 点击分组标题可切换；收缩时隐藏该分组所有标签项；默认全部展开 |
| F-04 | 标签选择 | 点击左侧标签项→右侧展示该标签分布详情 |
| F-05 | 搜索过滤 | 实时过滤左侧列表，匹配标签名或标签值（大小写不敏感）；右侧自动选中首个匹配 |
| F-06 | 搜索匹配高亮 | 当前选中标签中，匹配搜索关键词的标签值行背景高亮 |
| F-07 | 刷新 | 点击"刷新"→重新加载所有数据+更新时间 |
| F-08 | 更新时间 | 格式"更新：YYYY-MM-DD HH:mm:ss"，显示在顶部栏刷新按钮右侧 |
| F-09 | 非弹窗 | 页面独立路由，不通过弹窗或Modal承载 |
| F-10 | 数据范围自动判定 | 总部账号查总部数据，门店账号查分店数据，无门店选择控件 |
| F-11 | 默认选中 | 进入页面后自动选中第一个系统标签 |
| F-12 | 会员级别动态显示 | 会员级别标签值根据系统设置动态显示，不固定枚举 |
| F-13 | 性别"未设置"展示 | 性别标签展示"未设置"值；其他系统标签不展示"未设置" |
